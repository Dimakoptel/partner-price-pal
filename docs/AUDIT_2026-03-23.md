# Аудит системы MES COZY ART — 2026-03-23

## Выполненные доработки

### 1. ✅ Привязка настроек калькулятора к UI
- **Проблема**: `NumericInput` в `CalculatorForm` использовал захардкоженные min/max (например `max={300}` для опусков), игнорируя значения из `pricing_settings`.
- **Решение**: Все min/max ограничения в форме теперь читаются из `pricing` пропса динамически. Изменение `countertop_max_drop` в админке немедленно обновляет допустимый диапазон ввода.

### 2. ✅ Автосохранение расчётов
- Добавлена колонка `is_auto_saved` в `saved_calculations`.
- Каждый расчёт автоматически сохраняется с флагом `is_auto_saved = true`.
- Обычные пользователи видят только свои ручные сохранения. Админы видят все (включая авто).

### 3. ✅ Прайс-лист: ролевое разграничение цен + печать
- Менеджеры (admin или доступ к модулю clients) видят все 4 колонки цен.
- Остальные пользователи видят только РРЦ.
- Добавлена печать с выбором типов цен (чекбоксы: Дилер, Опт, Партнер, РРЦ).
- Печатная форма использует `logo_print_url` из настроек компании.

### 4. ✅ Тема-зависимые логотипы
- Добавлены настройки: `logo_light_url`, `logo_dark_url`, `logo_print_url` (категория branding).
- Боковая панель автоматически выбирает логотип по текущей теме.
- Убран фильтр `brightness-0 invert` — теперь используется оригинальное изображение.

### 5. ✅ Номенклатура: до 5 фото + галерея
- Добавлена колонка `photo_urls text[]` в таблицу `nomenclature`.
- Форма редактирования позволяет добавлять до 5 URL фото.
- При клике на фото открывается полноэкранная галерея с навигацией и миниатюрами.
- Обратная совместимость: старое поле `photo_url` синхронизируется с первым элементом массива.

## Рекомендации для дальнейшей реализации

1. **Загрузка фото через Storage** — заменить URL-поля на загрузку файлов в `company-assets` бакет.
2. **Персональный прайс для клиентов** — при входе клиента/партнёра/дилера показывать только его согласованную цену на основе `pricing_type` из таблицы `clients`.
3. **Cron для очистки авто-расчётов** — автоматическое удаление записей с `is_auto_saved = true` старше 30 дней.
4. **PDF-экспорт прайса** — генерация PDF через Edge Function для отправки клиентам.
5. **Аудит изменений прайса** — логирование изменений цен в `audit_log` для контроля ценообразования.
